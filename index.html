Note for imbedding a youtube video

Method 0 - It works - Fluid and Center

        
    <style>
    .video-wrapper {
	    position: relative;
	    padding-bottom: 40%;
	    padding-top: 0px;
	    height: 0;
    }
    .video-wrapper iframe {
	    position: absolute;
	    top: 0;
	    left: 15%;
	    width: 70%;
	    height: 100%;
    }
    </style>

    <div class="video-wrapper">
        <iframe src="https://www.youtube.com/embed/5RoteWQYHp8">
            <p>Your browser does not support iframes.</p>
        </iframe>
    </div>
  • Reference: https://stackoverflow.com/a/26346689/3811484

Method 1 - It works - Most Recommended

        
    <iframe src="https://www.youtube.com/embed/5RoteWQYHp8"
            style="display: block">
        <p>Your browser does not support iframes.</p>
    </iframe>
        
    
        
    <iframe src="https://www.youtube.com/embed/5RoteWQYHp8"
            style="display: block; margin: auto">
        <p>Your browser does not support iframes.</p>
    </iframe>
